About This System
What it does

Integrates AES-256 encryption with atomic electron orbital-based bit permutation to produce structured, visually interpretable ciphertext. Designed for both secure communication and cryptographic education.

Key Innovation

Dynamic S-boxes derived from SHA-256(key ∥ orbital_name) make permutations key-dependent — every password generates a unique permutation topology, eliminating static mapping vulnerabilities.

Security Stack

PBKDF2 (100,000 iterations) for key derivation · AES-256-CBC for encryption · Orbital S-box permutation for structured visual confusion · Base64 encoding for transport.

Research Context

Developed at MPSTME, NMIMS Mumbai. Targeting publication at IEEE INDICON / ICCCNT. Demonstrates cryptographic robustness via avalanche effect, Shannon entropy, and timing analysis.
